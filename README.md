## Development Tips

### Adding New Pages

1. Create a new HTML file in the `pages/` directory
2. Use the same HTML structure as existing pages for consistency
3. Link the main `style.css` and `script.js` files
4. Update navigation links on all pages to include the new page

### Styling Updates

- All styles should be added to `assets/css/style.css`
- Use CSS classes and IDs for styling
- Keep mobile-first responsive design in mind
- Test styles across different screen sizes

### JavaScript Enhancement

- Add functionality to `js/script.js`
- Use event listeners for interactivity
- Keep code modular and well-commented
- Test functionality across different browsers
